
**《梵经》**（Brahma Sutras），也称为《吠檀多经》，由婆多罗耶那（Badarayana）编纂。它是吠檀多学派的哲学经典之一，旨在对《奥义书》的教义进行系统的阐释和总结。通过简短而深奥的格言形式，这部经典对印度教的哲学进行了深入的探讨，尤其侧重于阐述梵（Brahman，至高存在）和我（Atman，个体灵魂）之间的关系，以及对宇宙、救赎和灵魂等重要哲学问题的回答。
